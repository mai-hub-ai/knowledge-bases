# 飞猪酒店套餐产品知识库 - Overview

## 产品定位

酒店套餐产品是飞猪酒店业务的核心产品形态，通过组合酒店住宿与增值服务（早餐、SPA、景点门票等），为用户提供高性价比的打包产品。

## 核心用户流程

```
频道页 → 酒店日历页 → 套餐详情页 → 套餐SKU浮层 → 下单页（普通/即买即约）
```

## 核心目标

- **D2O转化率**：提升套餐详情页到下单用户的转化
- **用户体验**：简化下单流程，降低决策成本
- **商业价值**：提升套餐GMV和用户复购
