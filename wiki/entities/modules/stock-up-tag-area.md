# 囤好货标签区

## 概述
囤好货标签区展示套餐的核心购买优势（先囤后约、不约可退、过期可退），是促进用户决策的关键模块。

## 核心属性

| 属性 | 值 |
|------|-----|
| 模块序号 | 7 |
| 模块类型 | 营销标签模块 |
| 位置 | 服务标签区下方 |
| 文档状态 | ✅ 已补充 |

---

## Slogan

### 当前Slogan
**先囤后约、不约可退、过期可退**

### 浮层解释文案

| 文案 | 解释 |
|------|------|
| **先囤后约** | 先下单锁定优惠，确定出行日期后，在线预约具体出行日期。 |
| **不约可退** | 若您的整份宝贝完全未进行预约，支持随时发起100%退款，整份宝贝部分预约则不支持退款。 |
| **过期可退** | 商品过期后您的整份宝贝仍未预约，系统自动100%退款，整份宝贝部分预约则不支持退款。 |

### Slogan变更位置
- summary存储的slogan
- 频道页（新老版本）
- 商品详情页
- 商详_加购浮层
- 商详_购买浮层
- 套餐下单页
