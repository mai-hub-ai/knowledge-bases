# 套餐图文详情区

> 原名：套餐权益描述区，已更名为套餐图文详情区

## 概述
以图文详情形式展示套餐的完整信息，包含商家亮点、图文详情、预约流程、预约说明、购买须知五个子模块。是用户了解套餐全部内容和规则的核心信息区。

## 核心属性

| 属性 | 值 |
|------|-----|
| 模块序号 | 11 |
| 模块类型 | 信息展示模块 |
| 文档状态 | ✅ 已补充 |

---

## 子模块结构

| 序号 | 子模块 | 数据来源 | 说明 |
|------|--------|---------|------|
| 1 | **商家说亮点** | [[summary]]（ic） | 商家配置的套餐卖点亮点 |
| 2 | **图文详情** | [[summary]]（ic） | 套餐图文介绍内容 |
| 3 | **预约流程** | — | 预约操作引导流程 |
| 4 | **预约说明** | [[summary]]（ic） | 预约相关规则字段（见下表） |
| 5 | **购买须知** | — | 购买相关条款和须知 |

### 预约说明包含字段

| 字段 | 说明 |
|------|------|
| 可入住日期 | 套餐可入住的日期范围 |
| 不可入住日期 | 不可入住的日期（如节假日限制） |
| 预约开放时间 | 预约功能开放的时间 |
| 提前预约天数 | 需提前几天预约 |
| 是否可拆分 | 套餐晚数是否支持拆分预约 |
| 补差价规则 | 不同日期的补差规则说明 |

---

## Constraints & Edge Cases
- 子模块 1（商家说亮点）、2（图文详情）、4（预约说明）的数据均来自 [[summary]]（ic）
- 预约说明的字段需与实际 summary 配置保持同步
