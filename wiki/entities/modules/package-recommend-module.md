# 套餐推荐模块

## 概述
套餐推荐模块是酒店日历页（小搜detail页）的套餐展示模块，分三个子模块展示不同类目的套餐商品。该模块的售卖转化率相较于其他渠道有明显优势。

## 模块位置
- **页面**：酒店日历页（小搜detail页）
- **位置**：页面中部/下部

## 业务背景

| 端 | ipvuv（日均值） | 支付uv（日均值） | 购买转化率（日均值） |
|---|---|---|---|
| 飞猪端 | 6602 | 438 | **6.65%** |
| 淘宝端 | 1187 | 44 | 3.74% |
| 支付宝端 | 841 | 39 | 4.7% |

## 选品规则

### 住宿套餐选品

根据以下规则选出套餐SKU list：

1. **类目ID过滤**
2. **SKU关联shid**：套餐SKU关联的shid需包含小搜传入的shid
3. **售卖有效期**：套餐为上架状态
4. **排除黑名单商品**

### 黑名单规则

满足以下任一条件的商品不透出：

| 条件 | 说明 |
|------|------|
| C卖家商品 | 暂无法识别，未做 |
| ictag=2614210 | 商品标签过滤 |
| 标题含违规词 | item标题或SKU标题包含违规词 |

**违规词列表**：
> 2人起订、单人、差价、补差、定金、咨询客服、单拍无效、确认后再拍、代订、代拍

### 定制逻辑

| 逻辑 | 说明 |
|------|------|
| 万豪集团 | shid只召回sellerid=3504342744 或 3470403533 的商品 |
| g35门店 | 套餐模块可透出酒景类目 |
| 模块不透出 | shid名单控制该shid下不透出套餐模块 |

## 排序逻辑

### 住宿套餐排序（SKU维度）

| 因子 | 权重 | 适用类目 | 备注 |
|------|------|----------|------|
| 业务加权标 | 20% | 全部 | ictag=2395202 |
| 类目=在线预约 | 10% | 在线预约 | - |
| cico时间可约 | 10% | 全部 | cico时间在可住日期范围内 |
| 大促商品标 | 10% | 全部 | - |
| 销量 | 20% | 全部 | item维度 |
| 价格（券后价） | 10% | 全部 | item的最低SKU价格 |
| 店铺优先级 | 10% | 全部 | 店铺名称包含"自营店"或"旗舰店"加权 |
| 商品首次上架时间 | 10% | 全部 | 新上架的商品优先 |

### 餐饮美食&玩乐套餐排序（Item维度）

| 因子 | 权重 | 适用类目 | 备注 |
|------|------|----------|------|
| 业务加权标 | 20% | 全部 | ictag=2395202 |
| 大促商品标 | 20% | 全部 | - |
| 销量 | 20% | 全部 | - |
| 价格 | 20% | 全部 | item的最低SKU价格 |

- 同item按SKU综合排序，最多透出1个满足条件的SKU

## 商品卡片信息

### 住宿套餐卡片（SKU维度）

| 字段 | 展示规则 |
|------|----------|
| **主图** | SKU1取商品第2张图，SKU2取第3张（若无则取第2张） |
| **标题** | 房型名称 + {晚数}晚 + {份数}份早餐 |
| **房型名称规则** | 单房型：直接展示；多房型："{房型名称}等可选" |
| **早餐份数** | =0时展示"无早餐" |
| **套餐内容** | X元素≥2时：共N项；X元素=1时：直接展示元素名称 |
| **标签** | 在线预约类目透出"囤好货"、"不约可退" |
| **卖家名称** | - |
| **价格** | 一口价（券后价） |
| **购买按钮** | - |

### 餐饮美食卡片（Item维度）

| 字段 | 展示规则 |
|------|----------|
| 商品主图 | - |
| 商品标题 | 优先短标题，无则用长标题 |
| 卖家名称 | - |
| 价格 | - |
| 购买按钮 | - |
