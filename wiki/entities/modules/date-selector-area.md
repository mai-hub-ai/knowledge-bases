# 日历页

> 原名：日期选择区，已更名为日历页。日历页是详情页的二级页面，分为即买即约日历页和普通日历页两种。

## 概述
日历页是用户查看可约日期、选择入住日期的二级页面。根据商品是否支持即买即约，进入不同的日历页：

| 条件 | 进入页面 | 核心差异 |
|------|---------|---------|
| item **支持**即买即约 | **即买即约日历页** | 可选择日期 + 下单，4 行日期布局 |
| item **不支持**即买即约 | **普通日历页**（旧版） | **仅展示**可住日历，不支持选择日期和下单，3 行日期布局 |

> 大促期间即买即约降级，此时走普通日历页数据（详见 [[日历组件]]）

## 核心属性

| 属性 | 值 |
|------|-----|
| 模块序号 | 8 |
| 模块类型 | 二级页面 |
| 切换字段 | `useNewCalendar`（服务端下发） |
| 文档状态 | ✅ 已补充 |

## 一、即买即约日历页（useNewCalendar = true）

### 入口
- 商详页「查看可约日期」
- 选酒店页「查看可约日期」
- 混合下单页「立即预订」

### 页面模块

#### 1. SKU 卡片
- 展示 sku 名称、售价、晚数
- 点击唤起套餐浮层
- 默认锚定：用户前置页面选择的 sku，若无则默认最低价 sku

#### 2. 房型卡片
- 展示房型名称、床型、窗户、面积
- 点击唤起房型详情浮层
- **排序规则**：优先可约天数占比高的房型 → 加价金额累计少的房型 → 随机

#### 3. 份数 & 间晚数选择
- 默认：购买份数=1，间数=1，晚数=sku 1份晚数
- 浮层可选购买份数、晚数、间数（联动）
- **购买份数**：
  - 可拆分：1～限购份数
  - 不可拆分：1～min（最大可约间数, 限购份数）
  - 限购份数 = min（item bbq 限购, 发布端限购）
- 仅支持**整份预约**，屏蔽非整份间晚数组合
- **不支持合并预约**（不展示同 sku 其他待预约订单的间夜数）

#### 4. 日历日期（4 行布局）

| 行 | 内容 |
|----|------|
| **Row1** | 今天/holiday 标签(20rpx) + "休"标签(黄底 #FFE033) |
| **Row2** | 日期数字 |
| **Row3** | 状态文案（可约/约满/紧张）+ 加价金额 |
| **Row4** | 升房/候补/预约有礼标签 |

- 加价 = rate 金额 - sku 凭证金额
- 日期表达：约满、紧张、可约、可升房（仅 0 元升房，过滤升房补差）
- **不支持**：候补库存、普通升房

#### 5. 预订总价（选中日期后）
```
预订总价 = 套餐总售价 + 总补差金额 - 套餐立减金额（活动）
套餐总售价 = 购买份数 × sku售价（summary 券后价）
购买份数 = 所选间夜数 / sku入住晚数
总补差金额 = Σ（rate金额 - sku凭证金额）
```

#### 6. 引导气泡
- **先囤后约引导**：未点击日期 + 停留 > 10s → 底部气泡引导进入先囤后约下单页
- **权益气泡**：用户选择日期后 → 展示立减 x 元（多个活动累加）

### 接口
- `mtop.trip.hotel.onlinebooking.calendar`（calendarScene=1）

## 二、普通日历页（useNewCalendar = false）

### 核心定位
- **仅展示**可住日历信息，**不支持选择日期和下单**
- 适用于不支持即买即约的商品

### 入口
- 商详页「查看可约日期」（item 不支持即买即约时）

### 日历日期（3 行布局）

| 行 | 内容 |
|----|------|
| **顶部行** | selectedNote > 升房图标 > "休"标签(橙底 #FF7300) > tExt 文案 |
| **中间行** | 节假日名/今天(28rpx) 或 日期数字(36rpx) |
| **底部行** | bExt（加价文案） |

#### tExt 颜色规则
| 状态 | 颜色 |
|------|------|
| 紧张 | 红色 #FF401A |
| 可升房 / 可协商 | 棕色 #805540 |
| 其他 | #5C5F66 |
| 置灰 | #CCCCCC |

### 接口
- `mtop.fliggy.traveldetail.onlinebooking.v2.calendar.query`（calendarScene=3）

## 三、新旧日历页核心差异

| 维度 | 即买即约日历页（新） | 普通日历页（旧） |
|------|-------------------|-----------------|
| **布局** | 4 行 | 3 行 |
| **功能** | 可选日期 + 下单 | **仅展示**，不可选择日期和下单 |
| **"休"标签** | 黄底黑字 #FFE033 | 橙底白字 #FF7300 |
| **节假日/今天** | Row1 独立展示，不替代日期数字 | 替代日期数字显示 |
| **状态文案** | Row3 独立行 dateDesc | 顶部行 tExt |
| **升房/候补** | Row4 独立标签 | 顶部行图标 |
| **加价** | Row3 与状态文案同行 | 底部行 bExt |
| **份数选择** | 支持（含整份约束） | 不支持 |
| **预订总价** | 展示并可下单 | 不展示 |
| **引导气泡** | 有（先囤后约 + 权益） | 无 |
